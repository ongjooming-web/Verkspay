ARCHITECTURE:

Device Detection:
  const isMobileDevice = /iPhone|Android/i.test(navigator.userAgent)
  const hasEthereum = typeof window !== 'undefined' && !!window.ethereum
  const shouldUseMobile = isMobileDevice || !hasEthereum

Mobile Flow (iOS/Android or no ethereum):
  1. User clicks "Connect Wallet"
  2. WalletConnect AppKit modal opens
  3. User selects wallet or scans QR code
  4. Deep link automatically opens wallet app
  5. User signs message in wallet
  6. Auto-returns to Prism app
  7. Address saved to Supabase

Desktop Flow (Desktop with window.ethereum):
  1. User clicks "Connect Wallet"
  2. MetaMask/Phantom popup appears
  3. User approves connection
  4. User signs message (optional)
  5. Address saved to Supabase

Message Signing:
  Method: personal_sign (EIP-191)
  Format:
    Sign in to Prism
    Wallet: {address}
    Timestamp: {Date.now()}
  Purpose: Authentication and replay attack prevention
