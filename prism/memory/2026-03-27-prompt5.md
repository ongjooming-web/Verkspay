## How It Works End-to-End

### Scenario: Invoice is 5 days overdue

1. **Cron triggers** (8 AM UTC daily via Vercel Cron)
2. **Query invoices** where status='unpaid' and due_date ≤ today and reminder_sent_count < 3
3. **For each invoice:**
   - Calculate `daysOverdue` = 5
   - Generate email reminder + send via Resend
   - Generate WhatsApp link: `https://wa.me/60123456789?text=...` (message includes "5 days overdue")
   - Store in reminders_log: `{ invoice_id, reminder_type: '3_day_overdue', whatsapp_link, ... }`
   - Increment reminder_sent_count on invoice
4. **User sees:**
   - Invoice detail page: "3 of 3 reminders sent"
   - Reminders card with two buttons: "📧 Email" and "💬 WhatsApp"
   - Click WhatsApp → opens wa.me link with pre-filled message
5. **User sends** WhatsApp message manually (or just shares the link)
