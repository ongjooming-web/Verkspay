# Session: 2026-03-27 — Recurring Invoices & WhatsApp Integration

## COMPLETED: Recurring Invoices (Prompts 1-3) ✅

### Migration
- Created recurring_invoices table with template support
- Fields: frequency, start_date, end_date, next_generate_date, status, invoices_generated
- Added FK to invoices table (recurring_invoice_id)
- RLS policies + performance indexes

### UI (Create, Edit, List, Pause/Resume/Cancel)
- `/invoices/recurring` — List page with table, status badges, actions
- `/invoices/recurring/new` — Create form with client, frequency, dates, line items
- `/invoices/recurring/[id]/edit` — Edit form with pre-filled data
- Pause/Resume/Cancel buttons with state management
- Empty states + responsive design

### Cron Job (Auto-Generation)
- `GET /api/cron/generate-recurring-invoices`
- Protected by CRON_SECRET header
- Finds active templates due today
- Creates invoice drafts with sequential numbers
- Calculates due dates from payment terms
- Updates next_generate_date based on frequency
- Auto-completes templates past end_date
- Returns generation summary
- Vercel cron: `0 8 * * *` (8 AM UTC)

**Setup Needed:**
- Add CRON_SECRET to .env.local and Vercel
- Already exists in Vercel (set Mar 19)
