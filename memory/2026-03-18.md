## UUID Error Details
- **Error**: "@supabase/auth-js: Expected parameter to be UUID but is not"
- **Location**: Settings page → Upgrade button click
- **Root cause**: Manual JWT decoding was unsafe and producing invalid UUID
- **Solution**: Use proper Supabase auth methods (`getUser()`, `getSession()`)
- **Status**: Fixed in client code; if still failing, check backend response
