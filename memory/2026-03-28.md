### Prompt 4 Part 1: AI Client Summary ✅
**Deployed commit:** 3b9e916

**Features:**
- `POST /api/clients/[id]/ai-summary` — Claude Haiku analysis
- Aggregates: invoices, proposals, recurring, notes, tags
- Calculates: payment speed, on-time %, win rate, client age, days since last invoice
- Plan gating: Pro (10/month), Enterprise (unlimited)
- Rate limiting with 30-day reset
- Master test account bypass
- Returns: Structured markdown + suggested tags (JSON array)
- Hook: `useAISummary(clientId)`

**Claude prompt:**
- System: Asks for 5 sections (Overview, Key Metrics, Observations, Recommended Actions, Suggested Tags)
- User: Structured data dump (client details, invoice history, payment behavior, proposals, recurring)
- Temperature: 0.3 (factual)
- Max tokens: 600
- Model: claude-haiku-4-5-20251001

**Data aggregated:**
- Client: name, company, email, phone, age (days)
- Invoices: count, total billed, total paid, outstanding, avg invoice
- Status breakdown: paid, unpaid, overdue, partial
- Payment: avg speed (days), on-time rate (%)
- Proposals: sent, accepted, declined, win rate (%)
- Recurring: active count, total per period
- Notes: last 5 recent
- Tags: current assigned

**Output parsing:**
- Saves full summary to `clients.ai_summary` + `ai_summary_generated_at`
- Extracts tags from `TAGS: ["tag1", "tag2"]` line using regex
- Returns both summary (markdown) + suggested_tags (JSON)
