# 2026-03-17 — Prism Phase 3 Wallet Integration Polish

## Session Summary
Refactored WalletConnect component to implement industry-standard mobile deep-linking + wallet browser detection. Scraped retry loop approach after discovering wallet browsers block programmatic `eth_requestAccounts` calls.

## Key Learnings (Critical for future wallet work)

**Wallet Browser Behavior:**
- MetaMask/Phantom mobile apps have built-in dapp browsers
- They inject `window.ethereum` asynchronously (need polling)
- They **block programmatic** `eth_requestAccounts()` calls from useEffect
- **Only user gestures** (tap/click) can trigger connection prompts
- Auto-triggering from useEffect silently fails or is ignored

**Industry Standard Approach:**
- Use URL parameters (`?autoconnect=true`) to signal user intent
- Show prominent "Tap to Connect" CTA when param + provider detected
- User taps button → wallet prompts connection
- This satisfies browser security requirement for user gesture

## Implementation Details (Commit: `51566d6`)

**Two Helper Functions:**
```typescript
isMobileDevice() — detects iOS/Android via userAgent
isInsideWalletBrowser() — checks for ethereum.isMetaMask/isPhantom + userAgent hints
```

**Wallet Browser Detection:**
- Polls `isInsideWalletBrowser()` up to 15 times (1.5 seconds total)
- Handles async injection of window.ethereum
- Sets `insideWalletBrowser` state for conditional rendering

**Three UX Paths:**
1. **Inside wallet browser** (MetaMask/Phantom mobile app)
   - Show: "✅ Wallet detected! Tap to Connect & Sign"
   - Button action: handleConnectWallet() → eth_requestAccounts → user approves
   
2. **Mobile without extension**
   - Show: "🦊 Open in MetaMask" + "👻 Open in Phantom" buttons
   - Action: Deep-link to wallet app with `?autoconnect=true`
   - MetaMask: `https://metamask.app.link/dapp/{domain-path}?autoconnect=true`
   - Phantom: `https://phantom.app/ul/browse/{encoded-url}?autoconnect=true&ref={encoded-url}`

3. **Desktop with/without extension**
   - With: Show "Connect Wallet" button → eth_requestAccounts
   - Without: Show error message to install extension

## Deep-Link Logic
```typescript
handleDeepLink(wallet) {
  const base = window.location.origin + window.location.pathname
  if (metamask) {
    href = https://metamask.app.link/dapp/{domainAndPath}?autoconnect=true
  } else {
    href = https://phantom.app/ul/browse/{encodeURIComponent(base)}?autoconnect=true&ref=...
  }
}
```
